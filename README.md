2. Commit - 3 modos de modelar una cabeza

	1. Box Modeling. Útil si no tienes necesidad de poner animaciones, porque pueden acabar en mal lugar los "Loops" importantes para esas cuestiones. Por lo que tendrías que recontruir tu malla para poder añadir detalles más adelante.
	2. Poly to poly. Te permite crear los "Loops" desde el principio y te da mayor control sobre el modelo, y para objetos complicados, como una cabeza, puede ser el mejor metodo de construcción.
	3. Sculpt & Re-Topologising. Es un método que suele consumir mucho tiempo pero te permite acabar con un modelo altamente detallado que mas tarde podria impreso en un modelo de LOD bajo, lo que da bastantes ventajas. (Modificador: Multiresolucion)
